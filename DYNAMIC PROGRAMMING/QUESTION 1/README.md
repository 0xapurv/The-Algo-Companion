<h2> The Tribonacci sequence Tn is defined as follows: </h2>

> T0 = 0, T1 = 1, T2 = 1, and 

> Tn+3 = Tn + Tn+1 + Tn+2 for n >= 0.
> 

Given **n**, return the value of **Tn**.

**EXPLAINATION:**


> **INPUT:** n = 4

> **OUTPUT :** 4

> **EXPLAINATION** :
> 
       T3 = 0 + 1 + 1 = 2
       
       T4 = 1 + 1 + 2 = 4
>
